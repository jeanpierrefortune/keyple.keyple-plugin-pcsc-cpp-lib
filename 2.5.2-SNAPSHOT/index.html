Keyple Plugin PC/SC C++ Library

Overview

The Keyple Plugin PC/SC C++ Library is a Keyple add-on that enables applications to interact with PC/SC compliant readers, supporting both contact and contactless smart card readers.

Key Features

  • Support for PC/SC standard readers
  • Contact and contactless card communication
  • Multiple protocol support (T0, T1, TCL)
  • Reader sharing modes (SHARED, EXCLUSIVE)
  • Direct control commands for proprietary features
